Troubleshooting: Tableau

Step-by-Step Guide: Tableau sometimes gives you code blocks that need a bit of formatting before they’ll render correctly. If your Tableau Public code isn’t rendering correctly on GitHub, follow these steps:

  1. Check the JavaScript code block in the Tableau embed snippet. Look for places where you can add **returns** after semicolons (;) or other logical breakpoints to improve formatting. A properly formatted code block looks like this: 
    <script type='text/javascript'>                    
  var divElement = document.getElementById('viz1580496195669');                    
  var vizElement = divElement.getElementsByTagName('object')[0];                    
  vizElement.style.width='100%'; 
  vizElement.style.height=(divElement.offsetWidth*0.75)+'px';                    
  var scriptElement = document.createElement('script');                    
  scriptElement.src = 'https://public.tableau.com/javascripts/api/viz_v1.js';                    
  vizElement.parentNode.insertBefore(scriptElement, vizElement);                
</script>
  2. Use the **Tableau Public share link** to embed an iframe instead. To do this:
    1. Copy the Tableau Public link from the Share button. Example: 
      https://public.tableau.com/views/CTSchoolDistrictsbyIncomeandGradeLevels2009-13/Sheet1?:language=en&:display_count=y&:origin=viz_share_link
    2. Delete all text after the question mark (?) in the URL. For example: 
      https://public.tableau.com/views/CTSchoolDistrictsbyIncomeandGradeLevels2009-13/Sheet1?
    3. Add the following parameters to the end of the URL: 
      :showVizHome=no&:embed=true
    4. Your final link should look like this: 
      https://public.tableau.com/views/CTSchoolDistrictsbyIncomeandGradeLevels2009-13/Sheet1?:showVizHome=no&:embed=true
    5. Enclose the link in an iframe with the following attributes: 
      <iframe 
  src="https://public.tableau.com/views/CTSchoolDistrictsbyIncomeandGradeLevels2009-13/Sheet1?:showVizHome=no&:embed=true" 
  width="90%" 
  height="500" 
  seamless 
  frameborder="0" 
  scrolling="no">
</iframe>
  3. Commit the changes and allow 2–3 minutes for the page to render on your GitHub.io portfolio.

Alternative Solution: If the above method doesn’t work, here are some troubleshooting tips:

  1. Ensure that the Tableau visualization is set to **Public**. Private visualizations cannot be embedded.
  2. Test the iframe link directly in your browser to confirm that it loads the visualization.
  3. Double-check that your repository is set up to serve HTML files. If it’s not, your visualization may not render.
  4. Use a static image or include a link to the Tableau visualization as a last resort: 
    <a href="https://public.tableau.com/views/CTSchoolDistrictsbyIncomeandGradeLevels2009-13/Sheet1?:showVizHome=no&:embed=true">
  View the Visualization on Tableau Public
</a>
