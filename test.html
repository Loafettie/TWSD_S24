Problem: Tableau is displaying the graph, but it is a static image and there is extra code?

Code Used:

<div class='tableauPlaceholder' id='viz1737946080854' style='position: relative'>
  <noscript><a href='#'>
    <img alt='General Government Debt to GDP Ratios' 
    src='https://public.tableau.com/static/images/Ge/GeneralGovernmentDebttoGDPRatios/Sheet1/1_rss.png' 
    style='border: none' />
  </a></noscript>
  <object class='tableauViz' style='display:none;'>
    <param name='host_url' value='https://public.tableau.com/' /> 
    <param name='embed_code_version' value='3' /> 
    <param name='site_root' value='' />
    <param name='name' value='GeneralGovernmentDebttoGDPRatios/Sheet1' />
    <param name='tabs' value='no' />
    <param name='toolbar' value='yes' />
    <param name='static_image' 
    value='https://public.tableau.com/static/images/Ge/GeneralGovernmentDebttoGDPRatios/Sheet1/1.png' />
  </object>
</div>
<script type='text/javascript'>
    var divElement = document.getElementById('viz1737946080854');
    var vizElement = divElement.getElementsByTagName('object')[0];
    vizElement.style.width='100%';
    vizElement.style.height=(divElement.offsetWidth*0.75)+'px';
    var scriptElement = document.createElement('script');
    scriptElement.src = 'https://public.tableau.com/javascripts/api/viz_v1.js';
    vizElement.parentNode.insertBefore(scriptElement, vizElement);
</script>

Solution: Replace the extra code with a simplified iframe to embed the active Tableau visualization.

Corrected Code:

<iframe 
  src="https://public.tableau.com/views/GeneralGovernmentDebttoGDPRatios/Sheet1?:showVizHome=no&:embed=true" 
  width="100%" 
  height="600">
</iframe>
