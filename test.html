Troubleshooting

Here is where I keep a record of all the issues people were having and how we might be able to fix them!

Problem: Tableau is displaying the graph, but it is a static image and there is extra code?

Steps to Fix:

  1. Locate the Tableau graph's original embed code, which is often in the format starting with a `
    ` block.
  2. Open the Tableau Public link to the visualization, usually included in the `host_url` or similar parameters. For example: 
    https://public.tableau.com/views/GeneralGovernmentDebttoGDPRatios/Sheet1
  3. Modify the URL to include these parameters for embedding:
    • ?:showVizHome=no to hide the Tableau header.
    • &:embed=true to make it an iframe-ready visualization.
  4. Replace the static block of code with this simplified iframe code: 
    <iframe 
  src="https://public.tableau.com/views/GeneralGovernmentDebttoGDPRatios/Sheet1?:showVizHome=no&:embed=true" 
  width="100%" 
  height="600">
</iframe>
  5. Test the iframe in your browser to ensure the visualization is interactive.

Solution: Use the corrected iframe code below:

<iframe 
  src="https://public.tableau.com/views/GeneralGovernmentDebttoGDPRatios/Sheet1?:showVizHome=no&:embed=true" 
  width="100%" 
  height="600">
</iframe>
